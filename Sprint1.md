## User Stories:
- As a new user, I want to be able to register for an account, so I can use the app and store my information.
- As an existing user, I want to be able to login to my account, so I can access the app's content.
- As a user, I want to be able to log my mass in my unit of preference (pounds or kg), so I can track my progress and incentivize continued efforts.
- As a user, I want to be able to log my height in my unit of preference (feet-inches or meters), so I can track height-dependent metrics like Body-Mass Index (BMI).
- As a user, I want to be able to calculate and log my BMI, so I have a fuller understanding of my health that's tailored specifically to my body.
- As a user, I want to be able to calculate my age in years from my DOB, so I can calculate and log age-dependent metrics.
- As a user, I want to be able to input my sex, so I can calculate and log sex-dependent metrics.
- As a more serious user, I want to be able to estimate my body-fat percentage (BFP), so I can obtain a deeper understanding of my health based on my BMI, age, and sex.
- As a user who bodybuilds, I want to be able to log the circumferences of my neck, waist, and hips, so I can track my gains and calculate my BFP more exactly.
- As a user who bodybuilds, I want to be able to calculate and log my exact BFP, so I can tailor my workouts and track my gains.
- As a user, I want to be able to visualize trends in my data, so I can better understand my progress (or lack thereof).
- As a user, I want to log my water consumption, so I can ensure I am hydrated.
- As a user, I want to track my calories consumed, so I can meet my daily goals.
- As a user, I want to track my macros (protein, carbs, etc.), so I can ensure I am consuming a balanced diet.
- As a fitness app user, I want to add and manage friends, so that I can connect with people I know and stay motivated through social support.
- As a fitness app user, I want to post updates and interact with my friendsâ€™ posts, so that I can share my fitness journey and engage with others for encouragement.
- As a fitness app user, I want to share my progress and achievements with my friends, so that I can stay accountable and motivated by social feedback.
